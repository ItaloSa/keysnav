# KeysNav

![Tests](https://github.com/ItaloSa/nav-js/workflows/Tests/badge.svg)


## Usage

- Vanilla

1- Import
 ``` html
 <script src="KeysNav.min.js"></script>
 ```

 2- Setup
 ``` html
 <script>
    const matrix = [['a', 'b', 'c'], ['d', 'e', 'f']];

    const keysnav = new KeysNav();
    keysnav.setGrid(matrix);
  </script>
 ```
